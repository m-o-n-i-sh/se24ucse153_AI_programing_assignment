# AI Programming Assignment
## Questions
This repository provides solutions and documentation for the following tasks:
1. **Turing Test & CAPTCHA:** Suggest and design appropriate architectures for implementing a Turing test and a CAPTCHA system.
2. **AQI Reflex Agent:** Design and develop a simple reflex agent that calculates the Air Quality Index (AQI) of a region based on environmental sensor data read from a mock file.
3. **Uninformed Search Strategies:** Implement BFS, DFS, and their variants. Formulate and solve classic AI state-space problems (Tic-Tac-Toe, Eight Queens, Milk Jug, Missionaries and Cannibals) and compare algorithm performance.
---
## Repository Structure
The codebase is organized into modular directories based on the assignment tasks:
* **`turing_captcha/`**
  Contains the architectural designs and programmatic simulations for the Turing Test and CAPTCHA challenge systems.
* **`aqi_agent/`**
  Contains the logic for the simple reflex agent (`aqi_agent.py`). It also includes a `data/` folder holding the simulated environmental sensor readings that the agent uses to determine the AQI.
* **`search_problem/`**
  Houses the reusable core implementations of blind search algorithms, including Breadth-First Search (BFS), Depth-First Search (DFS), Iterative Deepening, and Uniform Cost Search.
---
Do note that the second question has been added as a Git submodule as it was worked on in a separate repository as a group.\
